บทเรียนออนไลน์: โปรแกรมสำเร็จรูปอรรถประโยชน์ 1

รายวิชา ว20243 (ภาคเรียนที่ 1)

บทที่ 1: การจัดการเอกสารด้วย Google Docs

บทเรียนนี้จะช่วยให้นักเรียนเข้าใจหลักการทำงานของ Google Docs และสามารถสร้าง แก้ไข และจัดรูปแบบเอกสารได้อย่างมีประสิทธิภาพ เพื่อใช้ในการทำงานร่วมกันกับผู้อื่น

1. ส่วนประกอบและเครื่องมือของ Google Docs

ทำความรู้จักกับหน้าตาและส่วนประกอบหลักของ Google Docs ไม่ว่าจะเป็นแถบเมนู (Menu Bar), แถบเครื่องมือ (Toolbar), พื้นที่ทำงาน (Workspace) และแถบด้านข้างที่ใช้สำหรับแชร์หรือคอมเมนต์ การเข้าใจส่วนประกอบเหล่านี้จะช่วยให้เราใช้งานได้อย่างมีประสิทธิภาพมากขึ้น
